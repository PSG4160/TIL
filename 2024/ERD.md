User - Product: 1:N (한 유저가 여러 상품 등록)
User - UserFollower: M:N (UserFollower 테이블로 팔로우 관계 관리)
User - ProductLike: M:N (User가 여러 Product를 찜할 수 있고, Product도 여러 User에게 찜당할 수 있음)
Product - Comment: 1:N (상품 1개에 여러 댓글)
